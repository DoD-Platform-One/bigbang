# Package Lifecycle

This section organizes package work in Big Bang by lifecycle phase:

- [Onboarding](onboarding.md): Process-oriented planning and approvals needed before technical implementation
- [Integration](integration.md): Technical implementation steps to make a package Big Bang compatible
- [Offboarding](offboarding.md): Process and technical cleanup for removing or deprecating a package

If you are adding a new package, start with [Onboarding](onboarding.md), then move to [Integration](integration.md).
